1. [문제 인식 및 정의]
로그인 회원 가입에서 비밀번호 벨리데이션 검사를 안한다
2. [해결 방안]
	2-1. [의사결정 과정]
사실 모든 벨리데이션을 검사 안하고 있는데 비밀번호만 벨리데이션을 추가 할 지 고민 한다
	2-2. [해결 과정]
너무 힘들고 테스트도 바꿔줘야 할 수도 있을 것 같아서 그냥 비밀번호만 벨리데이션을 넣어준다
	
3. [해결 완료]
	3-1. [회고]
비밀번호 벨리데이션을 넣어주고 인증 관련 테스트를 수정해 준다
	3-2. [전후 데이터 비교]
이제 이상한 비밀번호로 회원가입과 로그인 을 할 수 없다 (근데 이전 데이터가 있으면 로그인은 벨리데이션을 넣지 말거나 비밀번호를 변경 하도록 유도 해야 한다 아니면 로그인 못한다)
